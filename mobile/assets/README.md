# App icons (ArtMuse)

Generated from `icon-source.png` (1024×1024 recommended).

| File | Size | Use |
|------|------|-----|
| `icon.png` | 1024×1024 | iOS / Expo app icon |
| `adaptive-icon.png` | 1024×1024 | Android adaptive foreground |
| `splash-icon.png` | 512×512 | Splash screen |
| `notification-icon.png` | 96×96 | Android notification (mono) |

Regenerate after replacing `icon-source.png`:

```bash
npm run app-icon
```

Configured in `app.config.js` → `icon`, `splash.image`, `android.adaptiveIcon`.
